This codes is an implementation of the Hall et. al paper and calculates the radial wave function from the 1s state and the 1s2s triplet state. 
The code is written for more general states but does not account for those wave functions not being orthogonal. 
The code fully works for the ground state of helium, however it fails to get the right energy for the ground state of the 1s2s triplet state. 
It does capture the right form of the wavefunction however.
